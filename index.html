Introducción a páginas web

🌐 ¿Qué es una página web?

  • Es una hoja digital
  • Se ve en el navegador
  • Contiene texto, imágenes y botones
  • Se construye con HTML, CSS y JavaScript

💻 ¿Dónde vive una web?

  • Vive en una computadora
  • Se llama servidor
  • Está conectada a internet

🖥️ ¿Qué es un servidor?

  • Computadora siempre encendida
  • Responde solicitudes
  • Entrega páginas web

☁️ ¿Qué es la nube?

  • Computadoras de otras personas
  • Acceso por internet
  • Realmente son muchos servidores

📞 Dirección IP

  • Identificador único
  • Como un número telefónico
  • Ej: 192.168.1.1

🔤 Dominio

  • Nombre fácil de recordar
  • Ej: google.com
  • Evita usar números

📚 DNS

  • Traduce dominio a IP
  • Funciona como agenda
  • Conecta nombre con dirección real

🔄 ¿Cómo funciona?

  • Escribes dominio
  • DNS busca IP
  • Se conecta al servidor
  • Se muestra la web

🧱 HTML

  • Estructura de la web
  • Define títulos y textos
  • Base de todo
<h1>Hola</h1>

🎨 CSS

  • Diseño y colores
  • Organiza visualmente
  • Hace la web atractiva

📁 CSS externo

  • Archivo separado
  • Más orden
  • Reutilizable
<link rel="stylesheet" href="estilos.css">

⚡ JavaScript

  • Lenguaje de programación
  • Da interactividad
  • Responde a eventos

📌 Resumen

  • HTML: estructura
  • CSS: diseño
  • JS: interacción
