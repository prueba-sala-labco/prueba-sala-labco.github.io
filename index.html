Introducción a páginas web

¿Qué es una página web?

Una página web es una hoja digital que ves en el navegador.

Se construye con HTML, CSS y JavaScript.

¿Dónde vive una web?

Una web vive en una computadora llamada servidor.

¿Qué es un servidor?

Es una computadora conectada a internet que responde a solicitudes.

¿Qué es la nube?

La nube es la computadora de otra persona a la que accedes por internet.

Dirección IP

Es la dirección única de una computadora.

Ejemplo: 192.168.1.1

Dominio

Es el nombre fácil de recordar de una web.

Ejemplo: google.com

DNS

Traduce dominios a direcciones IP.

Como una agenda telefónica.

Cómo funciona

Dominio → DNS → IP → Servidor → Página

HTML

Es la estructura de la página.

<h1>Hola</h1>

CSS

Es el diseño de la página.

CSS externo

Se guarda en un archivo aparte.

<link rel="stylesheet" href="estilos.css">

JavaScript

Lenguaje de programación que da interactividad.

Resumen

HTML: estructura

CSS: diseño

JS: interacción
